Example of Real-time Data API

This is the human-readable API specification for real-time data example of the DWBP document.

This example specification is taken from the UK Environmental Agency flood-monitoring API.

In this example the Transport Agency of MyCity keeps track of river flooding that may cause bridge closures during rainy seasons effecting bus rerouting and schedules. The API provides consumers real-time status information using a REST API. The API allows the consumer to select:

  • Current flood warnings and alerts
  • All station measurements
  • Latest flooding 3 day forecasts

API Description

Description API Parameters
Flood warnings and alerts {root}/data/readings min-severity, county, lat=x&long=y&dist=r
All station measurements {root}/id/stations/{id}/readings latest, today, date=d, startdate=d1&enddate=d2, _view=full, _sorted
Latest available forecast {root}/id/3dayforecast n/a
